# fetch-mini
让fetch使用更加简洁


你无需再进行繁琐的.then，现在，你只需要调用一个函数即可完成
-----------------
使用方法：   
request("链接","请求方式","响应体解析类型",function() {
})


示例：    
request("http://vhsf4d0eb40.lxoffice.work/","GET","json",data => console.log(data))
